# Final Year Project

This repository contains the core source code of final-year project on a wearable multimodal glove system for home-based rehabilitation and hand motor training.

Main folders:
- `final_code`: final deployed Python and Arduino code
- `Preprocessing`: labelling, window extraction, and preprocessing code
- `model_training`: model training code
- `model_offline_compare`: offline model comparison code
- `realtime_classifier_tests`: real-time classifier test code
- `Sensor_repeatability_test`: repeatability analysis code
- `rawdata`: raw collected data
